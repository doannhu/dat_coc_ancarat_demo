# Frontend Application

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS v4
- React Router v6
- Lucide React (Icons)

## Setup

1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```
